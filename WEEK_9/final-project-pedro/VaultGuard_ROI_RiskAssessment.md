  **2**   **Cohort non-renewal strategy**

  ------- ----------------------------------------------------------------------

+------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
| **How this works**                                                                                                                                                                                                                                                                                               |
|                                                                                                                                                                                                                                                                                                                  |
| Each safe runs an independent 5-year clock from installation. At each renewal window, VaultGuard has 60 days to serve written non-renewal notice. If not served, the safe auto-renews for another 5 years.                                                                                                           |
|                                                                                                                                                                                                                                                                                                                  |
| The strategy is not to cancel the entire warranty --- that is contractually impossible. It is to build a renewal calendar, identify which cohorts reach their window in the next 12--24 months, and make deliberate decisions: retain under warranty, transition to OOS-only, or switch to alternative hardware. |
|                                                                                                                                                                                                                                                                                                                  |
| This requires no agreement from SafeCore/ArmorGroup. VaultGuard exercises its contractual right to non-renew, one cohort at a time, backed by the data in this report.                                                                                                                                               |
